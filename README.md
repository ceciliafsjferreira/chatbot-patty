# Chatbot Patty

Este projeto consiste no desenvolvimento de um protótipo de chatbot com fins educacionais, criado para auxiliar estudantes no aprendizado da linguagem de programação Python.

O projeto foi desenvolvido com foco na prática de conceitos de desenvolvimento em Python, como a importação e utilização de bibliotecas, o desenvolvimento de interfaces interativas com o framework Streamlit e a integração com APIs.

##  Sobre o projeto

O chatbot **Patty** foi criado para auxiliar alunos durante o aprendizado da tecnologia Python, oferecendo uma interface interativa para tirar dúvidas e explorar conceitos relacionados à programação.

Além do desenvolvimento do chatbot, o projeto também tem como objetivo proporcionar prática em:

* Desenvolvimento de aplicações com Python;
* Utilização e importação de bibliotecas;
* Criação de interfaces interativas com Streamlit;
* Integração e consumo de APIs;
* Gerenciamento de sessões e histórico de mensagens;
* Organização de projetos Python;
* Utilização de ambientes virtuais (`venv`);
* Gerenciamento de dependências com `requirements.txt`.

## Tecnologias utilizadas

* **Python**
* **Streamlit**
* **APIs**
* **Git e GitHub**
* **Ambiente virtual (venv)**

## Estrutura do projeto

```text
chatbot-cec/
│
├── .venv/                 # Ambiente virtual

```

## Como executar o projeto

### 1. Clone o repositório

```bash
git clone URL_DO_REPOSITORIO
```

### 2. Acesse a pasta do projeto

```bash
cd chatbot-cec
```

### 3. Crie um ambiente virtual

```bash
python -m venv .venv
```

### 4. Ative o ambiente virtual

No Windows PowerShell:

```powershell
.\.venv\Scripts\Activate.ps1
```

### 5. Instale as dependências

```bash
pip install -r requirements.txt
```

### 6. Execute o chatbot

```bash
streamlit run Patty.py
```

Após a execução, o Streamlit disponibilizará uma URL local para acessar a aplicação pelo navegador.
